## Getting Started

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/haron-lee/my-stream-deck.git
   ```
2. Install NPM packages
   ```sh
   yarn install
   ```
3. Build project
   ```sh
   yarn build
   ```
